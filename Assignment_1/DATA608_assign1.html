Question 1

Create a graph that shows the distribution of companies in the dataset by State (ie how many are in each state). There are a lot of States, so consider which axis you should use. This visualization is ultimately going to be consumed on a portrait oriented screen (ie taller than wide), which should further guide your layout choices.

For all three questions, we are asked to give distributions of catagorical data: States or Industries. The simple Bar Chart seems to be the most intuitive way to present the information. I used horizontal bars to make the charts fit the portrait orientation better.

state_counts <- inc$State %>% table() %>% as.data.frame(stringsAsFactors
=FALSE)
colnames(state_counts) <- c('State', 'Frequency')


ggplot(state_counts, aes(x=reorder(State, Frequency),y=Frequency, color=State)) +
  geom_bar(stat='identity', color = 'black', fill=rainbow(52)) +
  coord_flip() + 
  xlab('US State')

Quesiton 2

Lets dig in on the state with the 3rd most companies in the data set. Imagine you work for the state and are interested in how many people are employed by companies in different industries. Create a plot that shows the average and/or median employment by industry for companies in this state (only use cases with full data, use Rs complete.cases() function.) In addition to this, your graph should show how variable the ranges are, and you should deal with outliers.

From the above chart, New York is in third place for high growth companies. I simply used tidyrs filter() function to subset the data. I used error bars to show the range of the variables. For the industries that have a single entry, and therefore undefined standard deviation, I imputed the value of 0 to draw the error bars.

#apadted: https://stackoverflow.com/questions/28687515/search-for-and-remove-outliers-from-a-dataframe-grouped-by-a-variable
ny_data <- inc  %>% filter(State == 'NY', complete.cases(.)) %>% arrange(Industry) %>% select(Industry, Employees) 
ny_data <- ny_data %>% group_by(Industry) %>% filter(!(abs(Employees - median(Employees)) > 1.5*IQR(Employees)))# Using 1.5xIQR as the outlier limit
